## 11. WHAT NOT TO DO

### Practices That Will Get You Penalized

| Tactic | Why It Is Dangerous | Google Penalty Risk |
|--------|-------------------|-------------------|
| **Private Blog Networks (PBNs)** | Google's algorithms detect PBN footprints (shared hosting, WHOIS data, linking patterns). A PBN penalty can tank your DA to single digits. | SEVERE -- manual action, deindexing |
| **Paid links (buying links from link sellers)** | Violates Google's Link Spam policies. Google's SpamBrain AI detects paid link patterns with high accuracy. | SEVERE -- manual action |
| **Link farms / link directories** | Low-quality directories that exist solely to sell links. Zero traffic, zero authority. | HIGH -- algorithmic devaluation |
| **Comment spam** | Leaving your URL in blog comments, forum signatures, or social media comments en masse. | MODERATE -- links devalued, possible spam flag |
| **Forum spam** | Creating forum accounts solely to post links. | MODERATE -- links devalued, accounts banned |
| **Link exchanges ("I'll link to you if you link to me")** | Small-scale reciprocal linking is natural. Large-scale link exchange schemes are detectable. | MODERATE -- algorithmic devaluation |
| **Article spinning** | Taking one article and "spinning" it into dozens of variations for submission sites. | HIGH -- thin content penalty |
| **Press release link stuffing** | Stuffing exact-match anchor text keywords into press releases. | MODERATE -- links devalued |
| **Fiverr/cheap link building services** | $5-$50 "500 backlinks" packages are 100% spam. They will hurt you. | SEVERE -- immediate damage |
| **Expired domain redirects** | Buying expired domains and redirecting them to your site. | HIGH -- algorithmic devaluation |

### Why These Are Especially Dangerous in 2026

Google's SpamBrain has undergone significant updates in 2025-2026. It now:
- Detects paid links with near-certainty using AI pattern recognition
- Identifies PBN networks by analyzing hosting, content, and linking patterns
- Devalues links from low-quality directories automatically
- Can apply algorithmic penalties without manual review (harder to recover from)
- Cross-references link profiles with known spam networks

### If You Get Penalized: Recovery Process

1. Run a full backlink audit with Ahrefs or SEMrush
2. Identify all spammy/unnatural links
3. Attempt to remove them by contacting webmasters
4. Create a disavow file and submit via Google Search Console
5. File a reconsideration request if you have a manual action
6. Wait 2-6 months for recovery (it is slow and painful)

**The bottom line:** Every shortcut in link building has a shelf life. The tactics in this document take longer but compound over time with zero penalty risk.

## APPENDIX C: ANCHOR TEXT STRATEGY

Maintain a natural anchor text distribution to avoid over-optimization penalties:

| Anchor Text Type | Target % | Examples |
|-----------------|----------|---------|
| Branded | 30-40% | "LDN Decks", "Loudoun Decks", "ldndecks.com" |
| URL | 15-20% | "https://www.ldndecks.com", "www.ldndecks.com" |
| Generic | 15-20% | "click here", "visit website", "learn more", "this company" |
| Topic/Natural | 15-20% | "deck builder in Northern Virginia", "composite decking contractor", "NoVA deck company" |
| Exact Match Keywords | 5-10% (keep low) | "deck builder Loudoun County", "composite deck cost Northern Virginia" |

**Rule of thumb:** If your anchor text profile looks unnatural (e.g., 50% exact-match keyword anchors), it will trigger a spam filter. Aim for the branded + URL + generic categories to make up 60-80% of your profile.
